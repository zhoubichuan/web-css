# 四.换肤

::: demo

```html
<style>
.black-theme { --bg: #171616; --fontColor: red; }
.white-theme { --bg: #f5f5f5; --fontColor: blue; }

.wrapper {
  backgroundColor: var(--bg);
  color: var(--fontColor)
}
</style>
<div id="box" class="white-theme" onclick="changeTheme()">
  <div class="wrapper">
    <div class="children2">123</div>
  </div>
</div>
<script>
  function changeTheme() {
    let ele = document.getElementById("box")
    if(ele.className ==="white-theme"){
      ele.className = "black-theme"
    }else{
      ele.className = "white-theme"
    }
  }
</script>
```

:::
